---
1. 특정 EC2에 ssh로 접속한다.
---
2. 특정 EC2 접속 후 해당 Instance에 해당 pem.key 권한을 넣고, 다시 접근 해준다.
```bash
chmod 400 {pem.key}
ssh -i {pem.key} ec2-user@{EIP Address}   #접근
```
---
3. 다시 접속 후 "password"를 지정해준다.
```bash
sudo passwd ec2-user
# New password
# 확인 password
```
---
4. 아래로 접근
```bash
sudo vi /etc/ssh/sshd_config
```
- 여기서 아래 내용을 추가한다.
```bash
PasswordAuthentication yes
AuthenticationMethods publickey,password
```
- 나와서 재시작을 해준다.
```bash
sudo systemctl restart sshd
```
