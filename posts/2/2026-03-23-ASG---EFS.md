### EFS
#### File System
- 사용자 지정으로 만들기
![](https://velog.velcdn.com/images/sorkdlrlsek/post/75fbca6e-37c7-460d-9a19-e21018214bdc/image.png)![](https://velog.velcdn.com/images/sorkdlrlsek/post/7ffbb545-a339-405b-8857-c0b2dea7ade8/image.png)![](https://velog.velcdn.com/images/sorkdlrlsek/post/1a1937b1-48ff-486a-a689-a5b4a59ac1a4/image.png)
- ASG에서 instance 등록 할 region 선택
![](https://velog.velcdn.com/images/sorkdlrlsek/post/1a0041d3-8d5c-41a1-8c5f-c76da433826a/image.png)
- 이렇게 선택하면 아래처럼 policy 나옴
```json
{
    "Version": "2012-10-17",
    "Id": "efs-policy-wizard-bba03776-c4e3-4d02-8779-b94ec790be18",
    "Statement": [
        {
            "Sid": "efs-statement-4ac01f8c-00a1-4f3d-b8dd-a4b38bc178b0",
            "Effect": "Allow",
            "Principal": {
                "AWS": "*"
            },
            "Action": [
                "elasticfilesystem:ClientWrite"
            ],
            "Condition": {
                "Bool": {
                    "elasticfilesystem:AccessedViaMountTarget": "true"
                }
            }
        },
        {
            "Sid": "efs-statement-9af56f20-b800-4252-a29c-e91714e604fe",
            "Effect": "Deny",
            "Principal": {
                "AWS": "*"
            },
            "Action": "*",
            "Condition": {
                "Bool": {
                    "aws:SecureTransport": "false"
                }
            }
        }
    ]
}
```
여기서 "elasticfilesystem:ClientMount" 만 추가
```json
{
    "Version": "2012-10-17",
    "Id": "efs-policy-wizard-bba03776-c4e3-4d02-8779-b94ec790be18",
    "Statement": [
        {
            "Sid": "efs-statement-4ac01f8c-00a1-4f3d-b8dd-a4b38bc178b0",
            "Effect": "Allow",
            "Principal": {
                "AWS": "*"
            },
            "Action": [
                "elasticfilesystem:ClientWrite",
                "elasticfilesystem:ClientMount"
            ],
            "Condition": {
                "Bool": {
                    "elasticfilesystem:AccessedViaMountTarget": "true"
                }
            }
        },
        {
            "Sid": "efs-statement-9af56f20-b800-4252-a29c-e91714e604fe",
            "Effect": "Deny",
            "Principal": {
                "AWS": "*"
            },
            "Action": "*",
            "Condition": {
                "Bool": {
                    "aws:SecureTransport": "false"
                }
            }
        }
    ]
}
```

sudo yum install amazon-efs-utils -y

mkdir /efs

echo "fs-XXXXXXXXXXXX.efs.ap-northeast-2.amazonaws.com:/ /mnt/efs efs _netdev,tls 0 0" >> /etc/fstab
