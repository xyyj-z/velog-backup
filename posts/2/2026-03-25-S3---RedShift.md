### 쿼리 편집기
![](https://velog.velcdn.com/images/sorkdlrlsek/post/28e37ada-5ec7-4a6c-b416-19bcfdab6d25/image.png)

- Table 생성

```sql
CREATE TABLE orders (
    order_id    VARCHAR(30),
    order_date  DATE,
    region      VARCHAR(30),
    product     VARCHAR(30),
    amount      INT,
    status      VARCHAR(30),
    customer_id VARCHAR(30)
);  
```
```sql
CREATE TABLE customers (
    id      VARCHAR(30),
    name    VARCHAR(30),
    region  VARCHAR(30),
    grade   VARCHAR(30)
);
```
![](https://velog.velcdn.com/images/sorkdlrlsek/post/6d162363-8b9d-42e4-9027-a021c15cd3a4/image.png)

- S3 가져오기
```sql
COPY orders
FROM 's3://2026-demo-redshift/orders/'
IAM_ROLE 'arn:aws:iam::515682373899:role/service-role/AmazonRedshift-CommandsAccessRole-20260325T102527'
CSV
IGNOREHEADER 1
REGION 'ap-northeast-2';
```
```sql
COPY customers
FROM 's3://2026-demo-redshift/customers/'
IAM_ROLE 'arn:aws:iam::515682373899:role/service-role/AmazonRedshift-CommandsAccessRole-20260325T102527'
CSV
IGNOREHEADER 1
REGION 'ap-northeast-2';
```
  - 확인 명령어
  ```sql
SELECT * FROM orders LIMIT 5;
   SELECT * FROM customers LIMIT 5;
  ```
