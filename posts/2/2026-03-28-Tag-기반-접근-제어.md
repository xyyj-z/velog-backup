## 특정 Tag만 S3 접근 허용

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:*",
      "Resource": [
        "arn:aws:s3:::<Bucket-name>/*"
      ],
      "Condition": {
        "StringEquals": {
          "aws:PrincipalTag/Name": "<특정-Name-Tag>"
        }
      }
    }
  ]
}
```

---

## **특정 Tag의 EC2만 시작/중지 허용**

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "ec2:StartInstances",
        "ec2:StopInstances"
      ],
      "Resource": "*",
      "Condition": {
        "StringEquals": {
          "ec2:ResourceTag/Name": "<특정-Name-Tag>"
        }
      }
    }
  ]
}

```

## 특정 Tag만 RDS 접근 허용

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": "*",
      "Action": "rds-db:connect",
      "Resource": "arn:aws:rds-db:ap-northeast-2:123456789012:dbuser:<DB-식별자>/<DB-user-name>
      "Condition": 
        "StringEquals": {
          "aws:PrincipalTag/Name": "<특정-Name-Tag>"
        }
      }
    }
  ]
}
```
