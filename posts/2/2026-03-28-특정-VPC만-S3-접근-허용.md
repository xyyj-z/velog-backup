---

- VPC Endpoint를 통해서만 S3 Bucket에 접근 가능
- Internet에서는 접근 불가

---

## VPC Endpoint를 이용

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Deny",
      "Principal": "*",
      "Action": "s3:*",
      "Resource": [
        "arn:aws:s3:::<Bucket-name>",
        "arn:aws:s3:::<Bucket-name>/*"
      ],
      "Condition": {
        "StringNotEquals": {
          "aws:SourceVpce": "<VPC-Endpoint-ID>"
        }
      }
    }
  ]
}
```
